### :memo: Week01_Familiar with MATLAB
#### Install MATLAB (for Linux)

$ sudo mkdir -p /usr/local/MATLAB/R2021a/

$ mkdir matlab

$ unzip -q matlab_R2021a_glnxa64.zip -d matlab (If need to install unzip -> $ sudo apt install unzip)

$ cd matlab

$ sudo ./install
